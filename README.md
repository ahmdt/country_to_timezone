# country_to_timezone

[![platform](https://img.shields.io/badge/Platform-Dart-02569B?logo=dart)](https://dart.dev/) [![platform](https://img.shields.io/badge/Platform-Flutter-02569B?logo=flutter)](https://flutter.dev) [![pub package](https://img.shields.io/pub/v/country_to_timezone.svg)](https://pub.dev/packages/country_to_timezone) 

Converts a ISO 3166-1 alpha-2 code to a timezone. (e.g. 'DE' to 'Europe/Berlin')

## Usage

Just import the package and use the `convert` function:

```dart
final timezone = CountryToTimezone.convert('DE'); 
// timezone = 'Europe/Berlin'
```
